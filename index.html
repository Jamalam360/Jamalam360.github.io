QuiltMC Mappings Triage

I help with triage for the mappings team of QuiltMC. The mappings team works on debofuscating Minecraft by mapping the unnamed decompiled code to new, named code. I work on reviewing pull requests of new mappings submitted by the contributors from within the community.

Mod Commisions for YouTubers

I've created many Minecraft mods for YouTubers, primarily SocksFor1 and RedFox . This involves being given a rough idea and developing it myself to create a fun and engaging mod for a video. It also involves working with other people such as texturers or modelers.

Personal Mods

I've created an array of Minecraft mods that add anything from simple blocks to an entire custom 'jam' creation system. Through this, I've gained over 500,000 downloads on CurseForge. You can find all my mods at CurseForge or GitHub . I also work with other people on my mods for textures, assets and code.

Discord Bots

I have created a variety of Discord bots that help users and moderators while they are using Discord. You can find all of my Discord bots on GitHub - the most notable one being Pinguino . I have also worked on Cozy for QuiltMC's community tooling team.

GitHub Actions

I am proficient in GitHub Actions, using it to automatically update my dependencies, find errors in my code, apply check styles and automate my project dependencies. Most notably, I have created a system to automatically deploy Pinguino to my VPS when an update is pushed to GitHub.

Deno and Web Development

I work with the Deno runtime using TypeScript. Some projects I have created include: Discord GitHub Releases, dog.jamalam.tech, an implementation of Bloom filters, a Link Shortener, a website to track companies that support NFTs, a Discord webhook library, a script that checks the height of a river to see whether it is low enough to row on, every morning. In addition, this website itself is 'compiled' using Deno - what you are reading right now was originally a Markdown file that has been substituded into HTML by my build-script. It also formats the HTML, and runs every time I push to GitHub. I've also have picked up some NGINX server skills from deploying all of my projects to my VPS.
